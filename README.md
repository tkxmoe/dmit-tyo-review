## TYO.T1 vs TYO.Pro：两条线路怎么选

DMIT 东京机房有两个产品线，T1 和 Pro，很多人搞不清楚区别。

TYO.T1 走的是 Premium Network，主要面向电信用户，CN2 GIA 回国，延迟和稳定性都很好。TYO.Pro 则是 CMIN2 线路，移动用户的回国体验更优，电信用户走 T1 更合适。

简单判断方法：你的宽带是电信或者联通，选 T1；主要是移动宽带，选 Pro。如果你不确定，T1 是更通用的选择，因为 CN2 GIA 对三网的兼容性都还可以，只是对电信最优。

价格上，T1 和 Pro 同配置下差距不大，主要差异在线路质量和适配人群，不是一个贵一个便宜的关系。
